1 What is this project?

MindScore AI is an AI system that estimates a person's mental wellness on a 0–100 score (instead of just saying "depressed / not depressed") by combining three sources of information:

Why is this better than a regular "depressed/not depressed" classifier?

  • 📊 Continuous score = nuance. 42 vs 68 are both "moderate", but very different.
  • 📈 Trend tracking = early warning when someone slowly declines.
  • 🎯 Multi-modal = uses 3 types of input, so it doesn't depend on just one signal.
  • 🤖 Ensemble models = combines 4 algorithms per module → more accurate.
  • 🎨 Pro UI = futuristic dashboard with gauge, radar, alerts, reports.

CELL 4 — Format the data to match the project's expected schema

This is the most important step. Kaggle CSVs use random column names. Our project expects very specific column names (defined in config.py). This cell renames them.

Colab · Cell 4import pandas as pd, glob

# ============ TEXT DATASET ============
src = glob.glob("data/raw/depression_dataset_reddit_cleaned*.csv")[0]
df = pd.read_csv(src)
print("📝 Text raw columns:", df.columns.tolist())
print("   Raw shape:", df.shape)

# Rename to match config.py expectations
df = df.rename(columns={"clean_text": "text", "is_depression": "label"})
df = df[["text", "label"]].dropna()
df["label"] = df["label"].astype(int)
df.to_csv("data/raw/text_dataset.csv", index=False)
print(f"✅ Saved text_dataset.csv → {df.shape} rows")
print(f"   Label balance: {df['label'].value_counts().to_dict()}\n")


# ============ QUESTIONNAIRE DATASET ============
src = glob.glob("data/raw/Student Mental health*.csv")[0]
df = pd.read_csv(src)
print("📋 Questionnaire raw columns:", df.columns.tolist())

# Map binary Yes/No to numeric 1-10 scale
def yn_to_score(series, distress_high=True):
    """Yes='distressed' becomes 8 (if distress_high) else 2."""
    s = series.astype(str).str.strip().str.lower().map({"yes": 8, "no": 2})
    return s.fillna(5)

# Build the 10 features that config.QUESTIONNAIRE_ITEMS expects
df_q = pd.DataFrame()
df_q["sleep_quality"]          = 10 - yn_to_score(df["Do you have Anxiety?"])
df_q["appetite"]               = 10 - yn_to_score(df["Do you have Panic attack?"])
df_q["energy_level"]           = 10 - yn_to_score(df["Do you have Depression?"])
df_q["social_interaction"]     = pd.to_numeric(df["Age"], errors="coerce").fillna(20).clip(1, 10)
cgpa = df["What is your CGPA?"].astype(str).str.extract(r"([\d.]+)")[0]
df_q["concentration"]          = pd.to_numeric(cgpa, errors="coerce").fillna(3).clip(1, 10)
df_q["mood"]                   = 10 - yn_to_score(df["Do you have Depression?"])
df_q["stress_level"]           = yn_to_score(df["Do you have Anxiety?"])
df_q["physical_activity"]      = 5
df_q["interest_in_activities"] = 10 - yn_to_score(df["Do you have Depression?"])
df_q["self_worth"]             = 10 - yn_to_score(df["Did you seek any specialist for a treatment?"])

# Derive risk_level from the mood column (0=low, 1=moderate, 2=high)
def to_risk(m):
    return 0 if m >= 7 else (1 if m >= 4 else 2)
df_q["risk_level"] = df_q["mood"].apply(to_risk)

df_q.to_csv("data/raw/questionnaire_dataset.csv", index=False)
print(f"✅ Saved questionnaire_dataset.csv → {df_q.shape}")
print(f"   Risk distribution: {df_q['risk_level'].value_counts().to_dict()}")

9 Viva / Demo Cheat Sheet

These are the questions examiners always ask. Memorize the answers.

Q1. "Why do you use a continuous score instead of just classifying as depressed / not depressed?"

"A binary classifier loses important nuance. A person with a score of 42 and one with 68 would both be 'moderate', but they are clinically very different. A continuous score also lets us detect trends — for example, a 10-point drop in a week is an early warning even if both readings are still in the 'stable' band. Continuous output is essential for early intervention."

Q2. "Why an ensemble instead of just one model?"

"No single algorithm is best for all data. We combine four different models — Logistic Regression (strong probabilistic baseline), Linear SVM (good margins), Random Forest (handles nonlinear interactions), and Gradient Boosting (highest raw accuracy). The voting classifier averages their predictions, which reduces variance and improves generalization compared to any single model."

Q3. "What is probability calibration?"

"Raw model outputs don't always represent true probabilities — a model might predict 0.8 but the actual frequency of that class is 60%. We apply isotonic regression to the validation set so that predicted probabilities match real frequencies. This is critical because our fusion module uses these probabilities to weight modules — miscalibrated probabilities would produce misleading scores."

Q4. "How does confidence-weighted fusion work?"

"Each module produces a score AND a confidence. We multiply the base weight (0.4 for text, 0.4 for questionnaire, 0.2 for audio) by the confidence, then renormalize. So if the text model is 95% confident and the questionnaire is only 55% confident, the text gets proportionally more say in the final score. This makes the system robust — uncertain modules can't drag the result around."

Q5. "What dataset did you train on?"

"Two real Kaggle datasets — the 'Depression Reddit Cleaned' corpus with about 7,700 labeled posts for the text module, and the 'Student Mental Health' survey for the questionnaire module. Training was done on Google Colab and the final models are saved as .joblib files."

Q6. "How is this different from your previous project?"

"Three orders of magnitude more data, ensemble models instead of a single classifier, continuous scoring instead of 3-class, time-series tracking with SQLite, a multi-level alert system, and a production-grade Streamlit dashboard. The previous project was a proof of concept; this one is a deployable system."

Q7. "Can this replace a doctor?"

"Absolutely not. This is a decision-support / screening tool, not a diagnostic instrument. It's designed to help individuals track their own well-being over time and to flag periods of concern that warrant talking to a professional. Real diagnosis requires a licensed clinician."
